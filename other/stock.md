# 常识
## 股票
***股票为什么有价值?***
```
股票代表了一个公司的所有权, 股票持有人(股东)可以在公司盈利后获取分红

🌙 分红金额直接存入券商账户
🌙 一般在业绩报告公布后发布分红计划, 但也有公司不分红(将盈利用于扩大生产)
🌙 每股分红X元, 对应的股价会在分红次日降低X元(XD除权)
🌙 现金分红可能需要纳税(券商代缴), 分红税计入"个人所得税", 持股未满一个月税率20%, 未满一年10%, 超过一年免税
```

***一级市场和二级市场的区别?***
```
🌟 一级市场 = 证券首次发行的市场, 相当于新房市场
🌟 二级市场 = 证券交易的市场, 相当于二手房市场

🌙 支付宝上购买的基金为一级市场, 直接向基金公司购买
🌙 沪深市都属于二级市场
```

***A股/B股/H股的区别?***
```
🌟 A股: RMB交易, 在中国注册的公司的股票, 流动性高, 如沪深北市
🌟 H股: HKD交易, 主业务在中国的公司的股票, 流动性中, 如港交所
🌟 B股: USD/HKD交易, 在中国注册的公司的股票, 流动性低

🌙 B股是历史产物, 现在随沪股通/深股通出现已边缘化
🌙 大陆散户可以通过港股通购买H股, 但有50W开户限制
🌙 除港股通外, 可以通过香港银行卡和香港券商进行港股交易, 但需注意中国外汇管制制度(5W美金/每人每年)
🌙 北向资金=通过沪/深股通流入A股的资金, 南向资金=通过港股通流入H股的资金
🌙 A股相较于H股一般有流动性溢价, 同一个公司的股票A股更贵
```

***股票交易时主要需要缴纳哪些费用?***
```
买卖股票都需要缴纳:
🌟 佣金 = 成交金额 * 0.05%, 券商收取, 买卖双方各0.025%

仅卖股票时缴纳:
🌟 印花税 = 成交金额 * 0.05%, 财税部门收取
🌟 股息红利差异扣税 = 分红金额 * 20%/10%/0%

仅基金需要缴纳:
🌟 基金管理费, 基金公司收取, 一般为0.5%/年
🌟 基金申购/赎回费, 基金公司收取, 一级市场买卖基金时收取

🌙 买卖一次股票需要缴纳的费用大致为成交金额的0.12%
🌙 二级市场交易基金时无印花税, 部分基金甚至没有佣金
```

***什么是定向增发股票?***
```
公司融资的一种手段, 以比二级市场更低的价格增加股票数量

🌙 增发股票一般认为对现有股东不利, 会稀释股东权益, 导致股价下跌
```

**什么决定了股票的价格?***
```
🌟短期: 供求关系, 买的人多价格涨, 卖的人多价格降
🌟长期: 公司的净利润和发展前景
```

***为什么降息会刺激股市?***
```
投资的沉默成本 = 银行存款利率, 降息会导致流动性从银行到股市/黄金
```

***A股的T+1/跌停/涨停机制?***
```
🌟 T+1机制: 当天买入的股票最早只能在下一个交易日卖出
🌟 跌停/涨停: 单个股票单交易日最大涨跌幅度有限制(10%或20%)
```

***什么是量化交易?***
```
采用技术(一般是深度学习)预测股票价格, 并根据预测结果进行交易

🌙 量化一般会挖掘互联网上的信息(政策+突发事件+股民舆情), 并在毫秒级别的延迟内根据信息执行高频交易
```

***什么是做空?***
```
在股价高点时从机构那里借股票卖出并支付一定利息, 在股价低点时买入股票还给机构, 从而获取盈利

🌙 A股不能做空, H股和美股可以做空
🌙 "平仓"指买入股票归还机构的过程, "强制平仓"指合约到期或当前亏损金额>保证金时, 机构强制买入股票
🌙 "空头"指做空的人, "券源"指借出股票的券商机构
```

***什么情况下股票会被退市?***
```
🌟 主动退市
🌟 连续两年净利润为负且营收低于1亿 => 强制退市
🌟 连续20个交易日股价低于1元/市值低于3亿 => 强制退市
🌟 重大财务造假/重大信息披露违法

🌙 有退市风险的股票会有*ST或ST前缀
```

***什么是IPO?***
```
IPO = 一家企业首次将其股份向公众公开出售

🌙 在IPO前, 企业的股票在创始人/风投公司/员工手里
🌙 在IPO后, 控股股东/风投公司的股票有限售期
🌙 散户参与IPO打新需要抽签, 中签才能买
🌙 A股的IPO为注册制, 只要信息披露合规就能上市, 对盈利能力和PE没有限制, 破发率高, 中签率低
```

## 基金
***股票型/混合型/债券型基金的区别?***
```
三种基金在投资对象比例上有差异:
🌟 股票型基金: >80%持仓为股票
🌟 债券型基金: >80%持仓为债券
🌟 混合型基金: 持仓比例介于股票型和债券型之间

🌙 因为持仓要求限制 即时熊市也不能空仓
```

***ETF和LOF的区别?***
```
两者都为开放式基金(可随时申购/赎回):
🌟 ETF: 被动跟踪特定指数或行业
🌟 LOF: 主动跟踪指数, 基金经理可主动管理

🌙 一般推荐买ETF, 统计发现LOF收益率不如ETF
```

***什么是基金净值和溢价/折价率?***
```
🌟 基金净值 = 每份基金份额对应的实际资产价值
🌟 溢价率 = 二级市场基金价格和基金净值的差异, 反应了二级市场的情绪

🌙 开放性基金的基金净值每个交易日结束后必须更新
```

***开放式基金和封闭式基金的区别?***
```
🌟 开放式基金份额不固定, 可随时找基金公司拿基金份额赎回现金, 也可以拿现金申购基金份额, 基金份额可以在二级市场交易
🌟 封闭式基金份额固定, 没有到期前不能直接找基金公司赎回, 基金份额可以在二级市场交易

🌙 封闭式基金一般折价率高, 到期后会停牌2~3个月, 然后会转为开放式基金(折价率收敛到0)
🌙 可以在封闭式基金停盘前买入折价率x%的基金, 停牌结束后吃折价率的方式获利, 假设基金的净值在停盘期间跌y%, 则停盘期间收益率=(x-y)%
```

***私募基金和公募基金的区别?***
```
🌟 发行对象不同: 私募只对高净值个人和机构发行, 公募基金可以对社会大众发行
🌟 流动性不同: 私募基金购买后在限定时间不能赎回(封闭型基金), 公募基金支持交易(开放型基金)
```

***货币型基金的作用?***
```
货币基金募集的资金用于短期高信用债券/银行存单/国债, 所以风险极低并且收益也低(但一般高于一年期定期存款利率)

🌙 一般来说, 货币基金在二级市场买卖手续费为0(T+0交易), 申购/赎回手续费为0, 视为现金等价物
🌙 货币基金是资金避风港, 支付宝的余额宝/微信零钱通都属于货币基金
🌙 货币型基金的"名义净值"基本不变(均为100), 采用均摊法计算利息
🌙 短融ETF和货币基金类似, 短融ETF用于投资短期债券, 收益略高于货币型基金
🌙 短融ETF和货币基金都适用于中短期求稳资金
```

## 期权
***什么是期权?***
```
期权 可以简单理解为 订金, 即花钱买一定日期后的某项权益
例如: 某投行发行的1元/张的期权为"1年后20元买入A公司股票的权利", 假设A公司当前股价为10元:
1️⃣ 如果一年后A公司股价小于等于20元, 那么期权内在价值为0, 收益率-100%
2️⃣ 如果一年后A公司股价为20.5元, 那么期权的内在价值为0.5元, 亏损0.5元, 收益率-50%
3️⃣ 如果一年后A公司股价为23元, 那么期权的内在价值为3元, 收益2元, 收益率200%

🌙 期权到了行权日时, 不必真实地执行权益, 一般可以兑换权益为现金
```

***什么情况下才能投资期权?***
```
期权风险极高, 只有风险和收益极不对称时, 才能投资, 且仓位必须<5%
```

***值得投资的期权的技术指标有哪些要求?***
```java
🌟 标的物(股价/指数)PE 小于 1
🌟 期权溢价率 小于 30%
🌟 期权剩余时间 大于 2年
🌟 期权杠杆倍率 高于 10倍

🌙 todo::2025Q3 资金小于100w前不要考虑期权, 上述技术指标的定义等有机会投资再补充
```

## 债券
***为什么说债券市场是更有效的市场?***
```
股票价格和公司价值往往有较大偏离, 债券市场基本没有(因为只需要考虑债券对应的公司违约/破案风险)
```

***债券为什么有价值?***
```
公司需要每年向债券持有者支付利息, 到期需要偿还本金+当期利息

🌙 债券的纸面利率通常是固定的(国债利率2-5%, 企业债券利率5%-12%), 但债券可在二级是市场交易, 成交价波动导致实际到期收益率会波动(实际到期收益率于成交价呈反比)
🌙 债券价格受通胀和基准利率影响, 价格和通胀率呈反比, 和基准利率也呈反比
```

***国债逆回购相较于货币型基金和短融ETF的优点?***
```
不受市场交易微小波动影响, 稳定盈利

🌙 建议超短期资金求稳可以做国债逆回购, 其余情况不建议投资普通债券
```

***什么是可转债?***
```
可转债相较于普通债券来说, 每年票面利息低很多(0.2%-3%, 每年递增), 但到达期限后可以将债券以固定的价格(转股价格)换成股票
例如: A公司发行3年可转债(面值100元), 每年利率为[0.5%, 1.5%, 2%+4%到期利率]， 转股价格为50元
1️⃣ 如果持有可转债到期后兑换后总收益为8%
2️⃣ 如果A公司当前股票价格为60元，转股后价值(100/50)*60=120元, 收益20%
3️⃣ 如果A公司当前股票价格为40元，转股后价值(100/50)*40=80元, 但只要不转股持有到期收益和1️⃣相等
4️⃣ 强制赎回: 如果A公司股价>=50*1.3=65元, A公司有权强行赎回可转债, 按照面值+利息赎回
5️⃣ 下调转股价格: 如果A公司股价<50*0.9=45元, A公司可以讲转股价格下调, 但必须高于每股净资产

🌙 本质上讲, 可转债 = 债券 + 期权, 具有债券的低风险属性和期权高收益机会
🌙 可转债发行的条件需要公司3年盈利且ROE和资产负债率满足一定要求, 风险低
🌙 可转债的债券面值固定为100元, 一般可转债在发行后半年后才能转股(转股起始日在发行日半年后)
🌙 强制赎回必须在转股起始日之后, 目的是逼迫可转债持有者转股(转股时收益为30%), 保护股东权益
🌙 根据市场统计, 多数转股债会被转股, 只有少数持有到期
🌙 可转债可以在二级市场交易, 实际交易价格90-200(理论130元就可以强制赎回, 但往往不会强制赎回)
🌙 A股买可转债需要2年交易时间资质
```

***什么情况下买入可转债?***
```
🌟 转股溢价率越低越好, 一般来说, 转股溢价率
🌟 价格越低越好, 一般来说, 110元以下买入收益高风险低
🌟 到期年化收益率越高越好, 到期年化收益率为负的不要买
🌟 股票波动率和潜力越高越好

🌙 上述结论, 完全不考虑公司的发展前景
🌙 假设当前转股溢价率=x%, 则需要股票价格涨x%才能不亏本
🌙 到期年化收益率 = 持有可转债到期后平均年化收益率, 是一个兜底指标
🌙 可转债的特性导致股票上涨收益能吃到, 下跌亏损吃不到
```

## 其他投资工具

***股票/黄金/实物资产/债券的风险/抗通胀能力/收益差异?***
```
风险: 股票 > 实物资产 ≥ 黄金 > 债券
抗通胀能力: 黄金 ≥ 实物资产 > 股票 > 债券
收益能力: 股票 ≥ 实物资产 > 黄金 > 债券

🌙 长周期看能产生正现金流的实物才算资产, 例如出租用的不动产(房/停车位), 负现金流的(车)算消费品不算做"资产"
```

***黄金/比特币等升值或贬值的逻辑?***
```
黄金和比特币都属于无现金流资产, 其升值逻辑/贬值取决于(稀缺性共识引起的避险属性+流动性变化), 有以下部分或全部特点:
🌟 价格 与 实际利率=(基准利率-通胀率) 呈反比
🌟 价格 与 美元信用和美元价值 呈反比
🌟 价格 与 战争风险 呈正比

🌙 非特殊情况或极大机会才考虑少量投资黄金, 因为没有办法衡量其价值(巴菲特从不投资黄金)
```


# 技术指标
***什么是市盈(PE)率?市净(PB)率?股息率?***
```
市盈率 = PE(Price/Earning) = 股票价格 / 每年净利润
市净率 = PB(Price/Book) = 股票价格 / 净资产
股息率 = Dividend/Price = 每股每年分红 / 股票价格

🌙 PE=6表示公司6年的盈利总额等于股价
🌙 PB=3表示公司净资产(总资产-总负债)的3倍等于股价, 注意轻资产公司(互联网公司)的PB一般偏高, 重资产公司(银行)的PB一般偏低
🌙 股息率=3%表示持有1年股票相当于存款年利率3%(不考虑股价波动)
🌙 不考虑成长性和衰退风险时, 股息率越高 & 市盈率越低 & 市净率越低 的股票越值得买入
```

***什么是PEG估值策略?***
```
PEG = 市盈率 / 净利润增长率, 一般来说PEG在0.5~1范围内是正常的, 高于1股票偏贵, 低于0.5股票便宜

🌙 例如公司净利润同比增长为20%, 按PEG估值策略, 公司合理的市盈率范围为10~20
🌙 PEG估值策略只对市盈率和净利润增长率为正的股票有效
🌙 PEG估值策略适用于非周期性行业, 在使用PEG估值时, 还需要考虑重资产/高负债/衰退预期等不利的因子
```

***什么是量比?换手率?***
```
量比 = 交易量 / 过去5天平均交易量
换手率 = 交易量 / 总流通股本 * 100%
```

***什么是负债率和市现率?***
```
负债率 = 总负债/总资产 * 100%
市现率 = 股票价格 / 经营现金收入 * 100%
🌙 负债是为了扩大生产, 完全没有负债的企业不一定是好企业
🌙 负债率过高(>=70%), 一般认为有高风险, 因为部分负债需要支付利息(公司账上现金不足以偿还有息负债的利息时会有流动性危机)
🌙 市现率过高意味着现金产出能力弱
```

***ROE/ROA的区别?***
```
ROE = 净资产收益率 (Return on Equity) = 净利润 / 净资产
ROA = 总资产收益率 (Return on Assets) = 净利润 / 总资产

🌙 投资界习惯用ROE衡量企业是否是一个好资产
```

# 财报阅读
## 简介
***阅读财报有什么用?***
```
财报能反映出经营状况不好的公司, 也能反映公司当前的风险和发展机遇

🌙 不要完全依赖财报, 但财报是投资很重要的依据
```

***财报三大表分别是什么?***
```
🌟 资产负债表
🌟 利润表
🌟 现金流量表

🌙 资产负债表最能体现公司的现状
🌙 除了三表外, 董事会报告也很重要, 反映了管理层水平
🌙 财报要横向和同行业其他公司对比, 也要纵向和公司往年(前5年)对比
```

***财报公布的最大延迟时间?***
```
🌟 一季报: 延迟一个月
🌟 中报: 延迟两个月
🌟 三季报: 延迟一个月
🌟 年报: 延迟4个月

🌙 超出时限未公布财报, 会被停牌处理
```

***什么是合并报表?***
```
合并报表是指母公司和其全部子公司形成的企业集团的财务报表, 需要抵消内部交易

🌙 如果子公司不是全资子公司(股权<100%), 需要在财报中标明"少数股东权益/损益", "少数股东"指除母公司外的股东
```

***什么情况下必须对半年报进行预告?***
```
🌟 扭亏为盈或扭盈为亏
🌟 净利润同比增速>50%或<-50%

🌙 如果不预告会退市
```

***什么是"标准无保留意见"?***
```
标准无保留意见 = 会计事务所认为财报没有问题, 除此之外的财报都有很大的造假风险
```

***净利润包含税收吗?***
```
不包含, 净利润为扣除企业所得税后的利润

🌙 中国企业所得税一般为25%, 但部分行业/企业税率更低或免税
```

***净利润可以全部用来分红吗?***
```
不能, 净利润的用途按优先级有:
🌟 弥补过往亏损
🌟 缴纳盈余公积金: 法律规定需要净利润的10%需要缴纳为盈余公积金, 这部分金额不能随意使用, 用于防范经营风险
🌟 分红
🌟 为未分配利润: 企业自由支配

🌙 盈余公积金累计金额超过注册资本50%时可以不用缴纳
🌙 未分配利润不代表公司的现金, 可能用于公司扩大生产/增加研发投入/并购竞争企业/偿还债务/回购股票
```

## 资产负债表
***无形资产有哪些?***
```
🌟 专利, 商标, 著作权等
🌟 软件, 域名等
🌟 特许经营权
```

***为什么农林行业容易财报造假?***
```
免税行业(虚构利润无需多交税), 资产价值难评估(会计难核查)的行业

🌙 如农林行业/集成电路/生物科技/医药科技行业
```

***什么是商誉(Good Will)?***
```
商誉 = 收购其他公司支付的溢价, 例如花100亿收购净资产80亿的公司100%股份, 那么多出的20亿就为商誉, 商誉=对被收购公司未来盈利能力的量化指标
```

***资产负债率如何衡量企业好坏?***
```
资产负债率过高(一般>70%)时, 认为企业负债过高, 可能因债务面临现金流危机

🌙 资产负债率低的企业抗风险能力更强
🌙 如果行业或企业在快速发展, 没有负债的公司不一定是好公司
```

***什么是计提?***
```
财报在计算资产价值时, 会对资产进行折旧, 以及对可能的坏账进行折合
```

## 利润表

***毛利率高说明什么?***
```
毛利高说明企业拥有定价权, 品牌价值高或技术壁垒高, 护城河强

🌙 典型高毛利企业: 茅台(高端), 腾讯(游戏), 爱美客(医疗器械)
```

***"扣非净利润"中的"非"指什么?***
```
"非"指"非经常性损益", 即偶发的损失或收益, 例如服装公司卖出厂房获取的利润, 这种损失或收益一般是不可复现的, 和经营无关的

🌙 通常 扣非净利润 比 净利润 更重要
```

***为什么不要用往年企业营收增长速率预估未来增长速率?***
```
🌟 单一市场是有限的, 例如茅台不会无限增长, 白酒市场已经到顶并且在萎缩
🌟 同时企业可扩展业务线, 例如阿里电商业务到顶后, 开始做阿里云

🌙 预估增速时需要综合多方面考虑
```

## 现金流量表
***为什么说净利润高不一定代表企业好?***
```
净利润高不代表企业现金流好, 因为可能销售款买方没有支付给企业, 但生产费用企业已经支付, 这样对企业的现金流不是好事

🌙 触发金融危机的美国雷曼公司市值7000亿美元, 但仅仅因为200亿美元的现金流无法筹集就破产
```

***现金流的分类?***
```
🌟 经营活动现金流: 如卖出商品获取的现金和支付员工工资付出的现金
🌟 投资活动现金流: 如企业购买生产设备的现金和购买公司股权支出的现金
🌟 筹资活动现金流: 企业发行债券借到的钱 和 企业偿还债务还的钱

🌙 主要关注经营活动现金流净额, 经营活动现金流净额 >= 净利润一般意味着公司产品卖的很轻松, 属于印钞机型公司
🌙 债务性筹资活动现金流过高意味着公司急迫的想要扩张, 及时承担债务
```

***什么是自由现金流?***
```
巴菲特提出的概念, 自由现金流 = 经营活动现金流净额 - 维持企业盈利能力的支出现金流

🌙 如果企业没有扩张, 自由现金流 ≈ 经营活动现金流净额 - 企业购建资产的现金
```

***如何通过现金流判断企业所属类型?***
```
🌟 经营现金流为+ 投资现金流为- 筹资现金流为-(奶牛型): 企业经营正常并且在稳步扩张, 而且可以回报股东
🌟 经营现金流为+ 投资现金流为+ 筹资现金流为-(母鸡型): 企业经营正常并企稳, 并开始回报股东和偿还债务
🌟 经营现金流为+ 投资现金流为+ 筹资现金流为+(妖精型): 企业计划大规模开启对内对外投资扩张 或 企业存在财务造假
🌟 经营现金流为+ 投资现金流为- 筹资现金流为+(蛮牛型): 企业急迫地进行投资扩张

🌟 经营现金流为- 投资现金流为- 筹资现金流为+(赌徒型): 经常出现在初创企业
🌟 经营现金流为- 投资现金流为- 筹资现金流为-(大出血型): 不建议投资
🌟 经营现金流为- 投资现金流为+ 筹资现金流为+(骗吃骗喝型): 不建议投资
🌟 经营现金流为- 投资现金流为+ 筹资现金流为-(混吃等死型): 不建议投资

🌙 奶牛型企业如果经营现金流可以覆盖投资&筹资现金流出, 是难得的优质企业, 例如13年的茅台
🌙 母鸡型企业, 如果低市盈率高股息率, 可以考虑买入
🌙 蛮牛型企业, 成王败寇, 如果筹资通过债务进行, 表明风险更大但是也表明企业管理层有信心, 如果筹资通过融资进行, 风险低但损害股东权益
🌙 赌徒型企业, 可以充分考虑发展前景适度参与
```

***如何通过现金流量表找到优质企业?***
```
🌟 经营现金流净额 > 净利润 > 0
🌟 投资现金流净额 < 0
🌟 现金及等价物增额 > 0
🌟 现金及等价物余额 > 有息负债

🌙 有息负债 = 短期借款 + 一年内到期非流动负债 + 长期借款 + 应付债券
```

# 实践经验

## 行业研究方法
***什么是防守板块?***
```
防守板块具有的特征: 收益稳定,波动小,高股息 常见的防守板块有:
🌟 黄金
🌟 银行
🌟 能源(水电煤油)
🌟 其他高股息股票
```

***什么行业有明显的周期性?***
```
行业中没有明显寡头, 并且产品无明显差异行业, 如金属 矿产 猪肉 煤炭

🌙 周期性逻辑: 竞争导致产能过剩 <=> 产能清出后利润上升 循环
🌙 宏观经济本身就是一个大周期, 与宏观经济绑定的海运行业也是周期行业
```

***如何分析一个行业?***
```
🌟 定价权
🌟 竞争格局: 决定了定价权
🌟 护城河: 决定了竞争格局
🌟 现金流安全: 保证公司稳定
🌟 规模效应: 是否有成本优势
🌟 政策

🌙 护城河因子: 成本/供应链优势, 技术壁垒, 垄断程度, 品牌忠诚度
```

***如何分析一个企业?***
```
🌟 企业的赚钱模式(高利润/高周转/高杠杆)
🌟 行业垄断程度, 与竞争对手的比较优势/劣势和竞争对手数量, 行业对潜在进入者的门槛
🌟 公司对上下游的议价权
🌟 公司当前的估值历史分位水平, 估值相较于同行业企业的估值水平
🌟 公司管理层对公司发展的影响

🌙 高利润值毛利率高的企业, 高周转指薄利多销型企业, 高杠杆指资产负债率高的企业
🌙 高利润看研发能力/差异化营销能力/产品力, 高周转看成本控制能力, 高杠杆看风控能力
```

## 交易策略
***为什么普通人不要做短线交易?***
```
短线交易会严重干扰正常工作生活, 且短线交易对普通人是不公平博弈
```

***投机和投资的区别?***
```
🌟投资: 成功的投资不一定需要预见未来, 更重要地是敏锐地发现市场中"风险和收益的不平衡"
🌟投机: 内幕消息+突发事件+市场情绪分析(心理博弈)+技术投机(K线/图标派)+流动性博弈

🌙 投机者也能赚钱(如索罗斯), 但从期望来说投资收益更高, 投机的年化回报率为[-7%, 9%], 投资的年化利率为[2%, 14%]
🌙 投资对社会发展更有益, 并且在中国股市政策影响大, 所以更推荐投资
```

***普通人的投资组合推荐?***
```
1️⃣ 3W现金保证紧急流动性
2️⃣ 半年生活费(5W)购买短融ETF
3️⃣ 50W+购买垄断性高股息股票(四大银行/高速/垄断型国企)
4️⃣ 在市场极度悲观时(黑天鹅事件, 例如上证指数<3000点)使用增量资金和3️⃣多余的部分买入指数
5️⃣ 在3️⃣的年股息收益能完全覆盖全年开支后, 可以考虑买入高风险的个股
```

***买股票要遵循的原则?***
```
🌟 永远不要想着暴富, 永远假设自己是傻子, 永远对市场心存敬畏
🌟 不要想赚到市场的每一份钱, 完美的T线可遇不可求, 适时离场锁定利润
🌟 不要加杠杆(任何情况不要贷款)
🌟 不要追高, 买在无人问津处, 卖在人声鼎沸时
```

***什么是烟蒂股策略?***
```
🌟 股价 < 每股净流动资产(流动资产 - 总负债, 即清算价值), 被市场低估
🌟 市值 < 100亿, 流动性差
🌟 资产负债率低, 现金流充足, 避免债务危机

🌙 烟蒂股策略来源于格雷厄姆, 施洛斯采用该策略年化15%
🌙 警惕烟蒂股财务造假导致ST风险
```

***什么是低估值分散化策略?***
```
🌟 根据指数的PE/PB, 在系统性机会时分散化买入低估值股票, 在系统性风险时减仓买入债券
🌟 维持股票类资产和债券类资产的仓位动态平衡(无论何时最少持有20%的债券类资产, 20%股票类资产)

🌙 根据历史统计, 指数PE小于10, PB小于1时为系统性机会
🌙 根据历史统计, 指数PE大于20, PB大于2时为系统性风险
🌙 分散化买入指买入多种行业的股票, 单行业仓位<15%, 单股票<7%
```

***低估值的股票有哪些特征时需要谨慎?***
```
🌟 公司将被时代淘汰(如: 诺基亚)
🌟 赢家通吃行业里的非龙头公司(如: 搜索引擎行业里的雅虎)
🌟 充分竞争无增长且重资产(夕阳行业)行业里的公司(如: 中国的钢铁公司)
🌟 周期性行业在周期顶点时
```

***应对突发事件的交易策略?***
```
🌟 突发利好事件不参与
🌟 突发利空事件: 分析是否对基本面造成影响(历史经验证明很多利空事件都无长期影响)

🌙 根据索罗斯"反身性理论", 利空事件往往会导致股票超跌, 突发利空事件时不要急于抄底, 需要考虑股价是否已充分反映利空
```
