# 简介
## 术语
**名义GDP和实际GDP的区别?**
```
名义GDP是不考虑通胀的GDP，实际GDP是考虑通胀的GDP。两者换算需要乘以GDP平准指数

🌙 如果不加以说明, 默认GDP指的是实际GDP
🌙 一般使用GDP衡量“经济水平”， 即社会有效总产出
🌙 GDP是重要指标, 但不能反映贫富差距/个人可支配收入/平均时薪
```

**CPI(Consumer Product Index)和PPI(Producer Product Index)的区别?**
```
CPI: 消费品(如手机,大米,猪肉)的价格变化率
PPI: 生产原料商品(如石油,钢铁,煤炭)的价格变化率

🌙 CPI和PPI可以一定程度上反映特定商品的通胀情况, 但通胀率=GDP平准指数, 需要考虑所有商品价格波动的加权平均值(权重=消费数量)
```

**什么是"大宗商品"?**
```
大宗商品是指在市场中可大规模交易的原材料商品, 例如: 原油,铝,玉米
```

**M0/M1/M2的区别?什么是M1-M2剪刀差?**
```
M0/M1/M2都是指货币, 区别在于三者流动性强度不同
🌟M0 = 纸币 + 数字人民币(DCEP), 广义上支付宝/微信余额也算M0
🌟M1 = M0 + 银行活期存款
🌟M2 = M1 + 银行定期存款
🌟M1-M2剪刀差 = M1的增速和M2增速的相对大小

🌙 M1增速>M2增速时, 活钱更多, 表明人们消费投资意愿强, 经济回暖
```

**名义利率/LPR/MLF/BP/实际利率的关系?**
```
🌟 1BP = 0.01%
🌟 MLF = 央行向商业银行贷款的利率, 货币政策工具
🌟 LPR = MLF + 银行加点利率
🌟 名义利率 = LPR + 合同固定加点利率
🌟 实际利率 = 名义利率 - 通胀率

🌙 MLF和LPR一般每月更新一次
🌙 如果合同为浮动利率, 则LRP变更会同步影响名义利率
```

**同比和环比的区别?**
```
基期不同, 同比会使用相同的年份为基期年份, 环比一般基期为上期(上月, 上季度)
```

## 常识

**实体经济和金融系统的关系?**
```
一个国家的经济系统由实体经济和金融经济两部分组成， 其中：

🌟实体经济包括服务业+制造业+农业等产业, 负责产出和消费, 通过财政政策调节
🌟金融系统包括央行+银行+证券市场+保险公司+基金公司+金融监管机构, 通过货币政策调节
```

**货币政策和财政政策的区别?**
```
🌟 货币政策: 作用于金融系统的政策, 常见的有降准降息, 定向降准降息
🌟 财政政策: 作用于实体经济系统的政策, 常见的有税收改革(税率调整+税种调整), 政府投资+转移支付, 国债和地方债

🌙 财政政策 = 政府怎么收钱和花钱的政策
```

**金融中介机构作用是什么？**
```
从资金闲置的对象（社保基金/个人/资本家）那里获取资金， 并将资金给需要融资的对象（企业/个人/政府），促进经济发展

🌙 融资方获取资金的方式有：债券，股权
🌙 商业银行，保险公司，基金公司，社保基金都属于金融中介机构
```

**货币的作用是什么？**
```
作为一般等价物提高交易效率（降低交易成本），从而促进经济发展

🌙 流动起来的“货币”才能促进经济，提高有效总产出
🌙 “货币”又称“流动性”， 其是一种流动性最高的资产（法律保证）
```

**流动性陷阱的成因?**
```
🌟 通缩预期: 企业和个人有通缩预期, 缩减借贷/投资/消费
🌟 风险厌恶: 投资风险高或社会保障低
🌟 债务过高: 企业因债务过高不愿意借贷, 个人因债务过高不愿意消费

🌙 流动性陷阱会导致货币政策失效, 即使0利率也无法刺激经济(货币流通不到实体), 并伴发失业率上升现象
```

# 宏观经济
## 凯恩斯理论
**凯恩斯理论的核心观点是什么?**
```
在生产力和生产要素充足的社会, 消费决定了经济发展水平
```

**决定消费水平的最重要因素是什么?**
```
收入水平

🌙 收入水平的增加对消费水平的促进有边际递减效应, 对储蓄水平的促进有边际递增效应
🌙 社会保障体系水平也会影响消费水平
```

## 通货膨胀
**什么是通货膨胀?**
```
通货膨胀指平均物价水平升高(同样的货币能买到的东西变少了), 通货紧缩指平均物价水平下降

🌙 CPI是衡量通货膨胀的重要指标, 对于普通居民来说比GDP平减指数更直观
🌙 温和的通胀有利于刺激需求, 一般认为同比CPI 2%-3%增幅为温和通胀
🌙 恶性通胀（通胀率大于50%）会导致货币信用危机，货币失去流动性, 进而交易成本变大，导致市场经济崩溃
```

**什么原因会导致通货膨胀(通货膨胀的分类)?**
```
🌟 货币超发: 货币增长率 > 有效生产增长率
🌟 成本推动型通胀: 生产要素价格的升高, 如石油危机, 工资涨幅过大
🌟 需求拉动型通胀: 社会总需求大于总供给, 如政府投资, 疫情期间政府发放现金补贴, 宽松货币政策
🌟 输入型通胀: 国际大宗商品价格上升
```

**通胀受益人有哪些?**
```
🌟 资产持有者有利
🌟 债务人(欠钱的人)

🌙 对应的通胀受害人为储蓄人, 债权人
🌙 通缩时, 上述利害关系相反
```

## 宏观经济政策
**宏观经济政策的目标是什么?**
```
🌟 充分就业(失业率)
🌟 通胀稳定(通胀率)
🌟 经济增长(实际GDP)
🌟 国际收支平衡(进出口额)
🌟 调节贫富差距
🌟 提高发展质量(产业结构/发展科技)
```

**什么是信号型政策?**
```
释放信号的政策, 可能并不实际实施, 用于改善人们的心理预期(信心是黄金!)
```

**为什么需要用紧缩型政策抑制需求?**
```
经济过热(需求远超供给)时:
🌟 产能过剩危机: 资金流向低效领域, 企业盲目扩张, 产能过剩不利于技术进步
🌟 通胀风险
🌟 债务危机: 负债率过高导致系统性经济危机
🌟 资产泡沫: 资产价格远高于实际价值, 没人接盘会导致金融危机
```

**什么是正回购和逆回购?**
```
🌟 正回购: 央行卖出证券/国债, 减少流动性
🌟 逆回购: 央行买回证券/国债, 增加流动性
```

**为什么央行不能随便印钞给政府?**
```
会造成恶性通胀, 丧失货币信用

🌙 政府发行国债一般需要在二级市场上过一圈(允许社会资金买入国债), 而不是直接让央行印钞收购
```

**货币政策工具有哪些?**
```
🌟 存款准备金率: 存款准备金率降低=释放流动性
🌟 再贴现率: 再贴现率降低=释放流动性
🌟 降息
🌟 公开市场业务: 如买卖国债

🌙 存款准备金最初目的是防止银行贷款过多产生的挤兑风险
🌙 再贴现率指未到期债券立刻换成现金的折价率
```

**央行如何干预汇率?**
```
🌟 在市场上买卖外汇
🌟 加息/降息
🌟 资本管制: 控制资本流动, 如对汇出资金征税

🌙 外汇交易为场外交易（一般为银行之间交易, 银行与公司和央行交易）
```

**为什么要干预汇率?**
```
避免汇率巨额波动时对经济发展造成影响, 如本币巨额贬值会导致资本出逃, 如本币巨额升值会导致出口受阻或产业外逃
```

**汇率变更对进出口的影响**
```
本币升值, 有利于进口; 本币贬值, 有利于出口

🌙 壁垒越高的行业中, 汇率对进出口的影响越弱, 例如汇率变更对芯片行业进出口影响小, 对石油/纺织行业影响大
```

## 中国宏观经济
**中国政府主要税种及其占税收收入的比例?**
```
🌟 增值税: 占比约30%, 企业卖出商品时缴纳, 税率约销售价格的10%
🌟 企业所得税: 占比约20%, 企业盈余净利润 * 25%
🌟 消费税: 占比约7%, 只对特定商品(烟,酒,奢侈品,珠宝)征收
🌟 个人所得税: 占比约7%, 打工人发工资或奖金时征收

🌙 上述数据取自中华人民共和国财政部2025年收支数据
🌙 当前房产相关税收(房产税+土地增值税+契税)总计占比6%, 财政收入中土地出让金约为房产税收的2倍
🌙 房产税=持有房产时缴纳; 土地增值税=转让土地使用权时缴纳; 契税=交易房产所有权时缴纳
```

**中国政府主要非税收入及其占比?**
```
🌟 政府性基金收入: 占比约70%, 主要是土地出让金(占政府性基金收入的80%), 其次是彩票
🌟 国有资产收入: 占比15%, 如矿产开采权/特许经营权/国企股权
🌟 罚没收入: 占比5%, 如交通违规/企业罚款

🌙 政府性基金收入专款专用(福利彩票用于社会福利, 土地出让金用于基建)
```

**中国中央政府和地方政府在收入分配的比例为多少?**
```
大部分税种 中央50%地方50% 或 中央60%地方40%
```

# 政策研究
## 医保政策
**职工医保和居民医保的区别?**
```
🌟 缴纳对象
职工医保： 在职职工
居民医保： 农村户籍+无业人员

🌟 缴纳方式
职工医保： 强制缴纳+一月一缴+单位和职工共同承担
居民医保： 自愿缴纳+一年一缴+政府补贴

🌟 缴纳年限
职工医保： 缴纳年限大于一定年限，退休后可享受权益，离职断缴3个月内可自费补缴
居民医保： 断缴后次年不享受权益

🌟 报销比例
职工医保：住院70%-90%
居民医保：住院50%-70%

🌟 账户区别
职工医保：有个人账户余额
居民医保：只有统筹账户

🌟 个人账户和统筹账户
个人账户：相当于存款， 医院/定点药店可以100%支付， 断缴后可正常使用, 合法情况下不能随意提现
统筹账户：有财政收入补贴， 并且报销有起付线和封顶线

🌙 报销时优先扣除个人账户余额再使用统筹账户
🌙 居民医保和职工医保不能同时使用， 但商业医保可以作为前两者的补充， 例如住院10W, 职工医保报销后需要支付3W, 3W可以再通过商业医保报销一部分
🌙 “新农合”=居民医保， 为历史上农村户籍的医保， 现已合并到居民医保中
🌙 看病时可以直接使用医保电子凭证， 免去后续报销流程， 但没有医保账户也可以看病
🌙 股市中的“社保基金”不包含医保
```

**如何看待中药/中成药企业?**
```
🌟 中药/中成药更偏向于消费品（药效存疑）， 但有很强的品牌溢价
🌟 零售占比30%， 院内占比70%， 但中药进医保审批宽松
🌟 主要受众为慢性病的老年群体
🌟 产品之间差异化较大， 不会存在高度垄断的企业
🌟 政策方面，对中药有支持

🌙 中药企业优先考虑： 品牌+销售能力+高溢价+养生类中药
```

**医药集采有哪些优点和缺点?**
```
医药集采指医保用药统一由相关部门集中采购，以量压价，公开招标
🌟 优点：节省了医保压力
🌟 缺点：药企需平衡成本和质量， 导致缺少精益求精动力

🌙 招标方式为暗标， 药企之间不知道对方的价格
🌙 集采客观上减少了医生的隐形收入和药企的销售支出，但需要相关部门抽检保证药效
🌙 集采一定程度上抑制了创新药的发展
🌙 集采制度实行时间：2020年
```

**创新药/仿制药的区别?**
```
创新药： 从研发到临床实验到批准上市的药物，
仿制药： 创新药20年专利保护期过后， 和创新配方相同的药物

🌙 创新药10年左右从研发到上市， 仿制药只要2年左右
🌙 仿制药药物成熟， 市场基本没增量； 创新药如果药效好是一个增量市场
```

**CRO/CMO的区别?**
```
为了鼓励创新药研发，政策审批时允许药品研发企业和生产企业分离， 其中药品研发企业=CRO, 药品生产企业=CMO

🌙 CRO代表企业药明康德， CMO代表企业凯莱因
🌙 CRO和CMO签订合约，在药品临床期间CMO支付一定费用，如批准上市后CMO可获得授权生产并销售
🌙 目前国内CRO授权国外CMO， 可获取大量授权费， 有利于创新药发展（国外创新药贵）， 但需考虑这种模式的政策风险
```
