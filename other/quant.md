# 简介
*量化交易和传统交易的区别?*
```
量化交易=西医, 传统交易=中医
```

*量化交易流程?*
```
1. 数据获取与清洗
2. 策略编写
3. 回测
4. 优化策略并继续回测
5. 模拟盘
6. 实盘自动交易
```

*常见的量化策略类型有哪些?*
```
🌟 多因子策略: 通过多个因子(特征)选股
🌟 传统技术策略: 均值回归策略/动量效应策略/二八轮动策略
🌟 机器学习策略

🌙 多因子策略是最主流策略
🌙 机器学习策略是黑盒, 并且容易过拟合
```

*常用的量化平台/框架?*
```
🌟 迅投QMT(本地运行)
🌟 PTrade(券商云端运行)
```

# 策略编写
*什么是未来函数?*
```
量化策略中使用了未来的数据作为交易依据或者假设一定能交易成功, 例如:
🌟 实盘中使用了当天设置之后的收盘价
🌟 假设在涨停能买入, 跌停能卖出
🌟 使用了当天的基金净值(基金净值需要等收盘后基金公司公布)
```

# 回测
*回测常见指标含义?*
```
🌟 基准收益: 用作比较的基准标的(如沪深300)的收益率。
🌟 Alpha (阿尔法): 策略超越基准的绝对超额收益。代表策略“跑赢大盘”的真实赚钱能力（越大越好）。
🌟 Beta (贝塔): 策略与大盘的联动波动性（系统性风险）。Beta=1与大盘同涨同跌，>1波动比大盘剧烈，<1波动比大盘平缓。
🌟 Sharpe (夏普): 风险调整后的收益（性价比）。衡量每承担1单位风险能换取多少超额回报（通常>1为优秀，数值越高收益越稳健）。
🌟 最大回撤: 账户净值从历史最高点跌至最低点的最大跌幅。数值越小越好
```

*为什么量化策略回测效果很好, 但实盘收益低?*
```
市场会变化, 策略也需要同步更新, 并且资金量大时会反向干扰市场导致策略失效

🌙 每个盈利策略都有资金容量, 超过资金容量alpha收益下降
```
