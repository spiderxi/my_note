## 2. 聊天和倾听

_和陌生人聊天应当聊什么?_

```
核心要点: 聊对方感兴趣的话题(家乡, 爱好, 优点, 职业)

* 家乡
你是哪里的人啊? | 四川那边有什么特别好吃的好玩的吗? 我打算有空去旅游

* 爱好(吃喝玩乐, 衣食住行)
你喜欢喝什么饮料啊? | 你喜欢吃什么菜啊? | 你喜欢玩什么游戏啊? | 你喜欢和别人一起旅游吗? | 你喜欢什么风格的衣服? | 你喜欢什么类型的男生啊?

* 优点
你是怎么找到女朋友的? 可以教我吗? | 你是经常健身吗? 我觉得你身材不错 | 我很喜欢你身上的香味, 你是用了什么香水吗

* 职业
你是老师吗, 教什么学科的? | 师傅做你们这行的是不是很辛苦啊? | 你们这一行是怎么挣钱啊, 我感觉很难挣钱 | 你们这个专业具体学什么啊?
```

_如何成为合格的听众?_

```
使用肢体和语言进行回应

>> 肢体: 点头 | 眼神直视对方
>> 语言: 询问谈话具体内容, 让对方说下去
```

_如何回应别人的赞美?_

```
感谢 + 自谦或自夸
>> "谢谢, 我其实成绩一般, 没有那么好"
>> "开玩笑, 我的人格魅力是你能够想象的吗?"
```

_如何回应别人合理的批评?_

```
承认缺点 + 表达上进心
>> "我其实也觉得我自己表达能力很差, 我最近也在看人际交往的书, 可惜还没有学会"
```

_为什么不要一昧地赞美会让人反感, 而匹配别人的小错误却对增进关系?_

```
* 一昧地赞美对让人认为不真诚

* 批评别人的小错误会让别人认为你更懂他们
```

# 3. 亲密关系

## 1. 建立关系

_如何建立和维系亲密关系?_

```
* 经常关心他人
>> 认同感需要

* 支持他人并给予帮助
>> "陪伴是最长情的告白"
```

_为什么偶尔让别人帮你小忙可以促进关系?_

```
帮助他人会让他人认为你认同他的价值
```

_更有吸引力的人有什么特点?_

```
1. 外表好看的人
>> 进化学解释: 外表好看的人财富和地位较高, 繁殖能力更强

2. 和自己相似的人
>> 和自己相似的人相处更容易预判对方的行为, 焦虑感更低

3. 经常看见和相处的人(日久生情)
>> 进化学解释: 原始以部落为单位生存

4. 喜欢自己的人
>> 进化学解释: 更容易从喜欢自己的人获取帮助
```

_为什么"得不到的在骚动"?_

```
人会对得不到的事物有过度积极的想象, 但在得到后会因为想象的落差感到过度失落
```

_第一印象为什么难以改变?_

```
人类更倾向于印证自己认知的正确性(自尊)
```

_如何拒绝他人的表白?_

```
直接拒绝!!!
>> 只要拒绝表白对别人的伤害都是一样的, 所以请直接说出"抱歉, 我不喜欢你"
```

## 2. 沟通

_为什么说恶意中伤(言语侮辱/嘲讽/阴阳)并不能产生任何积极影响?_

```
恶意中伤唯一的作用是缓解自己身心上的痛苦, 但并不能让他人认识和改正错误
```

_如何批评别人?_

```
肯定别人 + 采用问句指出错误
>> 你这首歌旋律很抓人耳朵, 但缺少创新, 我觉得你是不是可以在歌里多加一些自己的东西? (批评口水抄袭歌)
```

_沟通问题时为什么要使用第一人称替代第二人称?_

```
* 第二人称: 暴露他人的问题引起反感, 适得其反
>> 你就不能爱干净点吗?

* 第一人称: 先表达自己的感受, 再提出问题
>> 我觉得寝室里很脏很乱, 要不你整理一下你的卫生吧?
```

_为什么复述别人的观点可以让沟通更好?_

```
* 复述可以让别人知道自己在认真倾听

* 复述可以暴露双方之间的认知差异
```

## 3. 维持关系

_为什么一次破坏关系的行为(例如吵架)影响力远大于一次增进关系的行为(一起旅游)?_

```
损失厌恶
```

_为什么"婚姻是爱情的坟墓"?_

```
结婚后伴侣往往不能满足自己的期望

tip: 不要对自己的伴侣和爱情有过高的期望, 不要认为理所应当
```

_爱情由哪三部分组成?_

```
激情 亲密 承诺

* 激情
由性激情和超出预期(惊喜)组成

* 亲密
由两个人的相似程度决定

* 承诺
>> 履行给予对方物质支持和精神关爱的责任
>> 保持对伴侣的忠诚
```

_冷战和冷暴力为什么不会让对方意识到错误和后悔?_

```
冷战只会让别人陷入迷茫和困惑, 如果要表达不满请直接沟通
```

_造成家庭暴力和吃醋的原因是什么?_

```
自信不足(往往是能力不足导致自信不足)
```
